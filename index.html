Project Terbaru

Berikut adalah project-project yang pernah saya buat

Onime (Public API Service)

Layanan API yang menyediakan data anime dengan token Otentikasi. Respon yang diberikan dari layanan ini seperti judul anime, total episode, pengarang, tanggal rilis, deskripsi, rating, genre, dan lain-lain. Seperti API publik pada umumnya, layanan ini juga memiliki dokumentasi berbasis situs web. Pengguna juga dapat berlangganan VIP dengan mengunjungi website untuk mendapatkan token VIP. Project ini juga sudah dicover dengan feature test.

Teknologi : Laravel, PHP, MySQL, JQuery, Bootstrap, Snap-Midtrans, Laravel-FFMpeg/FFMprobe.

Shalltears Clothing Store

Website yang memberikan layanan kepada Shalltears Clothing Store, untuk membantu menentukan desain produksi dengan sistem voting. Desainer Terdaftar dapat berkontribusi, situs web juga membantu toko menjual produk mereka. Dibangun dengan Laravel dan terintegrasi dengan External Payment Gateway (Midtrans).

Teknologi : Laravel, PHP, MySQL, JQuery, Bootstrap, Snap-Midtrans

Shoes Clean

Situs web yang ditujukan untuk pemilik bisnis Laundry Sepatu, sehingga mereka dapat mendaftarkan bisnis mereka. Proses pemesanan menggunakan sistem booking dengan pembayaran langsung pada saat sepatu diserahkan ke toko, sehingga customer tidak perlu bingung untuk memesan.

Teknologi : Laravel, PHP, MySQL, JQuery, Bootstrap

Aplikasi Bumdes (SAD MERTHA NADI)

Aplikasi yang digunakan untuk mengelola dan menginput data pembelian, penjualan dan laporan barang pada Bumdes Sad Mertha Nadi.

Teknologi : Laravel, PHP, MySQL, JQuery, Bootstrap
